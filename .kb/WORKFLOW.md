# Workflow Notes

## TortoiseGit

This project uses **TortoiseGit** on Windows. TortoiseGit requires explicit add/remove operations for new and deleted files (similar to SVN).

When adding new files: use "Add" from the context menu, or `git add`.  
When deleting files: use "Delete" from the context menu, or `git rm`.  
Modified files are detected automatically.
